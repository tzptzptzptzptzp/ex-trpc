## 紹介

こちらは React / express を使用した TodoList です。  
tRPC を使用し、効率的にデータの取得を行なっています。

## 使用技術

- [React](https://react.dev/)
- [tRPC](https://trpc.io/)
- [Node.js](https://nodejs.org/ja)
- [express](https://expressjs.com/ja/)

## 開発手順

npm を使用しています

```bash
// バックエンド起動
npm
cd backend
npm run dev

// フロントエンド起動
npm
cd frontend
npm run dev
```
